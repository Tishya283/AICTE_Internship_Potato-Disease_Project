#The Potato Disease Prediction System is an AI-powered application that detects and classifies potato leaf diseases using deep learning. It helps farmers take timely action by analyzing leaf images and identifying conditions like Late Blight, Early Blight, or Healthy status. Built with Python, Flask/Django, TensorFlow/Keras, and OpenCV, the system offers a user-friendly web interface for real-time disease detection.
